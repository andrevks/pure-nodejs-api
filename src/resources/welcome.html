Welcome to Pure NodeJS API

This project is a comprehensive Node.js API implementation showcasing intermediate skills in pure JavaScript development.

It's a RESTful API designed to manage tasks. It provides various endpoints to perform CRUD (Create, Read, Update, Delete) operations on tasks.

GET /tasks: Retrieve a list of tasks.
POST /tasks: Create a new task.
PUT /tasks/:id: Update a task by ID.
DELETE /tasks/:id: Delete a task by ID.
PATCH /tasks/:id: Complete a task by ID.
POST /tasks/upload-csv: Upload tasks from a CSV file.

For more information, check out the GitHub repository.
